# Dibble v3.5 — GPU Acceleration + Live Point Cloud Preview Design

## 1. GPU Acceleration

### What
COLMAP supports CUDA acceleration for SIFT feature extraction, matching, and dense MVS. Currently all commands run on CPU. Add optional GPU acceleration.

### Changes
- `PhotogrammetryConfig` — add `use_gpu: bool = False`
- Expert GUI — add "Use GPU acceleration" checkbox with GPU info display
- COLMAP subprocess calls — append `--SiftExtraction.use_gpu 1` etc.
- Pipeline startup — detect GPU availability and show card name

## 2. Live Point Cloud Preview

### What
During processing, show the evolving point cloud / mesh at each stage so the user sees progress visually.

### Changes
- Progress page — embed a small PyVista viewer widget (reuse Viewer3D pattern)
- QTimer — polls COLMAP output dir for new point cloud files every 2s
- On each stage transition, show that stage's output in the preview
